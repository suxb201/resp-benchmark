# RESP Benchmark

`resp-benchmark` is a tool designed for testing databases that support the Redis RESP protocol.

## Features

1. **Efficient**: This tool can generate high test load with low resource usage.
2. **Customizable Workload**: It allows for user-defined workloads, which is useful for continuous integration (CI)
   testing.

## Usage

```bash
resp-benchmark <CONFIG>
```

### Arguments

- `<CONFIG>`: The input file to use.

### Options

- `-h, --help`: Print help information.
